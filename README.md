# My-Blog
My personal blog code

## Installation

1. `python -m venv env`
2. `source env/bin/activate`
3. `git clone https://github.com/sosolidkk/my-blog.git`
4. `cd my-blog`
5. `pip install -r requirements.txt`

## Usage

1. `export FLASK_APP=run.py`
2. `flask db init`
3. `flask db migrate`
4. `flask db upgrade`
5. `flask run`
